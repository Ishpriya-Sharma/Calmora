# 🌿 Calmora

A full-stack mental wellness web application that allows users to securely write, manage, and track personal journal entries.

---

## ✨ Features

- 🔐 User authentication (Register/Login)
- 📝 Create and manage journal entries
- 📦 RESTful API for data handling
- 💾 MongoDB database integration
- 📱 Responsive and simple UI

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  
- **Authentication:** JWT (can be added/improved)
